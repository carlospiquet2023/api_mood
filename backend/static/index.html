Ajuda - Como Usar

1. Upload do Arquivo

Faça upload de um arquivo ZIP contendo os diplomas em formato PDF. O sistema irá extrair e validar automaticamente os arquivos.

2. Posicionamento do QR Code

Clique na posição exata onde deseja que o QR Code apareça em todos os diplomas. Use o primeiro diploma como referência.

3. Processamento Automático

O sistema irá processar cada diploma, extrair o nome do estudante, buscar dados no Moodle e inserir o QR Code personalizado.

4. Download dos Resultados

Baixe o arquivo ZIP com todos os diplomas processados e os QR Codes inseridos.
